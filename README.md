## 🎯 Objetivo do Repositório
Este repositório contém exclusivamente o backend do projeto BaratieFit. Seu objetivo é disponibilizar uma API responsável por gerenciar as regras de negócio, autenticação, persistência de dados e comunicação com os serviços utilizados pela aplicação.

## 🛠️ Tecnologias Utilizadas
- Node.js — Ambiente de execução JavaScript utilizado para construir a API.
- Express.js — Framework responsável pela criação das rotas e gerenciamento das requisições HTTP.
- Knex.js — Query Builder utilizado para conexão, consultas e manipulação do banco de dados.
- MySQL — Sistema de gerenciamento de banco de dados relacional utilizado para armazenar as informações da aplicação.
- JWT (JSON Web Token) — Implementação de autenticação baseada em tokens.
